Stampout

Renter-owned deposit evidence packets. Guided walkthrough, timestamped PDF, no install.

Screens

  1. 01 - Home (landing) Hero, trust bullets, dual CTA. Ref: Craft, Fireflies (Mobbin)
  2. 02 - Create report Move-in/out, state, address. No account
  3. 03 - Room list Progress + room cards. Ref: Chime setup, Liven (Mobbin)
  4. 04 - Capture room Photo grid, area chips, issue tagging. Ref: Outlook Real Estate (Mobbin)
  5. 05 - Review Flags, summary. Ref: Manus task progress (Mobbin)
  6. 06 - Export PDF preview, download, email. Ref: Alan, Chime (Mobbin)

Core flow

Landing -> Start report
  -> Move-in | Move-out + address
  -> Room list (presets)
  -> Capture (wide shot -> issues -> OK)
  -> Review (flags)
  -> PDF export -> success
